# Place Value

This folder contains three simple Python programs:

1. **number_in_words.py**

   - Input a number from 1 to 100
   - Output: Number written in words (e.g., 45 → "forty-five")

2. **expanded_form.py**

   - Input a 3-digit number (100-999)
   - Output: Expanded form (e.g., 456 → 400 + 50 + 6)

3. **round_number.py**
   - Input a number from 1 to 100
   - Output: Number rounded to the nearest 10

## Folder Structure

number_tools/
│
├── number_in_words.py
├── expanded_form.py
├── round_number.py
└── README.md

## How to Run

Open terminal, navigate to this folder, then run:

```bash
python number_in_words.py
python expanded_form.py
python round_number.py
Enter the number when prompted and the output will be displayed.
```
